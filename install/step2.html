Step 2: Install Certificate

By adding the certificate, we finish the routing of traffic to the Trailers application to PopcornTV.

  1. Scroll down to where it says "Send Data to Apple" and select it then set it to "No"
  2. Once your setting says "No", while hovering over "Send Data to Apple" (like shown in the image below) press the "Play/Pause" button on your Apple TV remote. It will open up a screen that we will use in the next window.


  3. Select "Add Profile" then click "Ok"
  4. On the "Add Profile" page it should pull up a text box. Type in http://trailers.apple.com/trailers.cer then click "Submit". This adds a custom profile to your Apple TV that allows it to connect to PopcornTV.


  5. Exit out to the Applications home page
